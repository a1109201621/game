## 在聊天互动界面添加"特化"按钮,预设口交,足交,腋下,小腹,手交特化.同时添加自定义按钮,让用户输入要求,然后根据以下模板生成特化添加到界面.同样也是只能选择xl/max和apex三个模型(带一个小提示:此处对模型要求较高,不推荐使用低智商模型):
```
使用yaml格式输出<xx特化>,用以指导ai进行创作,每个特化不要超过500中文汉字!!!

格式:
```yaml
<xx特化>
特化内容
</xx特化>
```
```yaml
<xx特化>
特化内容
</xx特化>
```
```
特化可以无上限添加,但是超过三个时需要弹出警告:因为ai上下文窗口限制,过多的特化会增加积分的消耗
